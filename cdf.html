Cory’s “Defense First”

Systematic Trading Strategy

Strategy Overview

I first saw this stratgey on AllocateSmartly's website. Carlson’s “Defense First”
What's cool about the original strategy is that it uses defensive assets to guide the risk posture of the portfolio, only holding risk assets when defensive assets are not exhibiting strong momentum.
My version follows the same outline but with different instruments and a different momentum factor.

Logic & Design

The strategy is to measure the momentum of the defensive instruments and compare them to the momentum of the "Risk-Free- Rate."
In this case, I'm using the ETF BIL (State Street SPDR Bloomberg 1-3 Month T-Bill ETF) as the "Risk-Free-Rate."

Higher in momentum means higher allocation percentages 40%, 30%, 20%, 10%.

The Defensive instruments I've chosen are;
BNDX (Vanguard Total International Bond ETF)
TLT (Ishares 20+ Year Treasury Bond ETF)
DBC (Invesco DB Commodity Index Tracking Fund)
GLD (SPDR Gold Shares)

If a defensive instrument does not exceed the risk-free-rate, the offensive instrument receives it's allocation.
The Offensive instrument is:
QQQ (Invesco NASDAQ Trust)

Key Characteristics

Defense First is a monthly rotation strategy. I coded this strategy in Python because the account it's running on is a smaller account and Python provides a clean way to trade using fractional shares.
At this point, it's only been live for a month. But I've built, tested and watched this strategy for months in RealTest and I'm happy with it's performance thus far.

Performance Notes

As this account grows, Defense First will be it's core allocation.
This TAA strategy should keep the account protected in times of stress while providing respectful growth.
I'm looking forward to watching this one.
